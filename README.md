## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/reponame.git
cd reponame
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

The app will run at `http://localhost:5173`.

## Customization

### 1. Personal Content

Open `src/config.js`. This file contains all the text, links, and data for the website.

- **Personal Info**: Update names, titles, and location.
- **Social Links**: Add your GitHub, LinkedIn, etc.
- **Experience & Projects**: Add your own history and portfolio items.

### 2. Images

Replace the project images in `src/config.js` with your own URLs or import local images.

### 3. Styling

The project uses Tailwind CSS. You can customize colors and fonts in `src/index.css` or `tailwind.config.js` (if added, or via Vite config).
